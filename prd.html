1. 제품 비전 및 포지셔닝

🎯 Product Vision

"Bring AI to Every Call" — 금융, 보험, 이커머스의 모든 고객 통화를 AI가 처리하거나 보조하는 범용 플랫폼

포지셔닝

기존 모델: SI/구축형 (단일 고객사, 맞춤형 엔진, 장기 프로젝트)

신규 모델: 범용 멀티테넌트 SaaS (다수 고객, 표준화된 기능, 빠른 온보딩)

핵심 가치: "규제 준수 + 자동화" — 금융 규정은 지키면서, 이커머스 속도감은 살린다

  • 기대값: 3개월 내 도입, 리스크 제로

    • 2. 속도형 (이커머스) — 김민지 리더

    • CX팀 리더, 30대 여성, 스타트업 경력, PM 마인드

    • 기대값: 1개월 내 POC, 6개월 내 본격 운영

      • 3. IT 의사결정권자 — 박진호 임원

      • IT/디지털전환 임원, 45+ 세대, 인프라 중심
      • 관심사: 멀티테넌시 보안, SLA, 운영 비용
      • 의사결정: 벤치마크, 장기 로드맵, 계약 조건
      • 기대값: 운영 자동화, 기술 부채 해소

        • 3. 핵심 기능 정의 (기술 컴포넌트)

        📌 중요: 제품 A/B/C는 "기술 컴포넌트"입니다

        다음 장의 가격 모델 (Starter/Business/Enterprise)은 이 기술 컴포넌트들을 조합한 "비즈니스 패키징"입니다.

        • Starter 티어: A 보이스봇만 포함
        • Business 티어: A 보이스봇 + B Agent Assist (기본) + C 온보딩
        • Enterprise 티어: A + B (고급) + C + 감정분석, CRM연동, 맞춤개발 등 전체 기능

        제품 A: AI 보이스봇 (자동 응대)

        기능 설명 기술 자산
        실시간 음성 처리 고객 음성 → STT (200ms) → LLM (400ms) → TTS (300ms) + 네트워크/오버헤드 (600ms) = 약 1.5~1.8초 총 지연 보이스봇 코어 엔진
        대화형 지능 도메인 특화 LLM, 다중 턴 문맥 유지, 자연스러운 음성 톤 Fine-tuned LLM
        Closed-loop 태스크 수행 고객 의도 파악 → 사내 API 호출 → 실행 결과 전달 (예: 주소변경, 환불 신청) Agentic AI + Tool Invocation
        지식 기반 검색 고객 문의 → 하이브리드 RAG (Elasticsearch 키워드 + Neo4j 그래프) → 정확한 정책 인용 Elasticsearch + Neo4j
        감정 기반 에스컬레이션 실시간 감정 분석 → 부정 임계값 초과 시 자동 상담원 전환 감정 분석 모델

        제품 B: Agent Assist (상담원 보조)

        기능 설명 효과
        실시간 트랜스크립트 고객-상담원 통화 자동 녹취 및 텍스트 변환 상담원이 메모 중 놓치는 내용 없음
        감정 분석 UI (Phase 2에서 추가) 고객 감정 점수 (-1.0~1.0) 실시간 표시, 부정 전환 시 경고 상담원의 대응 톤 즉시 조정 가능
        동적 배틀카드 고객 문의 내용 자동 분석 → 관련 정책/지식 추천 AHT 23% 단축, 신입 온보딩 기간 1/2 감소
        통화 요약 통화 종료 후 2초 내 자동 요약 (주제, 해결 여부, 후속조치) ACW(After Call Work) 제거, 실시간 CRM 업데이트
        고객 정보 패널 호출자 신원, 계정 이력, 진행 중 문제, 구매 정보 한눈에 개인화된 상담 가능, 교차판매 기회 증가

        제품 C: 고객사 온보딩 (셀프 서비스)

        • 회사 정보 입력 (산업, 도메인, 월 통화량)
        • 봇 페르소나 정의 (이름, 인사말, 톤, 언어)
        • 지식 업로드 (PDF/URL, 자동 임베딩)
        • 샘플 통화로 테스트 (시뮬레이션)
        • → 3~5일 내 실제 운영 시작

        4. 기존 자산 활용 계획 및 신규 개발 범위

        보유 자산 현 상태 SaaS 전환 신규 개발
        보이스봇 코어 엔진 구축형 프로젝트에서 검증됨 멀티테넌시 격리 추가 테넌트 컨텍스트 관리
        Elasticsearch + Neo4j RAG 단일 고객사 벡터DB/그래프DB 테넌트별 인덱스/DB 격리 동적 인덱스 생성 API
        프롬프트 관리 패널 내부 개발팀용 테스트 도구 LLMOps 대시보드로 승격 고객 접근 권한, UI 개선
        React AdminPanel 내부 구축 프로젝트용 제품 수준 콘솔로 개선 다중 고객사 관리, 계정 시스템
        고객용 채팅/음성 인터페이스 웹/모바일 양용 가능 그대로 재사용 브랜딩 커스터마이징
        Agent Assist 화면 없음 (신규) — 완전 신규 개발 필수
        온보딩 플로우 없음 (신규) — 완전 신규 개발 필수
        멀티테넌시 보안 개별 프로젝트 수준 — 아키텍처 레벨 필수 (DLS, JWT 검증)

        5. 기능 우선순위 (MVP vs Phase 2 vs Future)

        📍 MVP (0→1 달성, M1~M3, 3개월)

        목표: 범용 SaaS 플랫폼의 기본 형태 완성 + 첫 고객사 온보딩

        • 멀티테넌시 데이터 격리 (Elasticsearch DLS, Neo4j 별도 DB, JWT 인증)
        • 보이스봇 코어 SaaS 래핑 (호출 방식 표준화, 과금 미터링 기초)
        • Agent Assist 기본 기능 (트랜스크립트, 지식 추천)
        • 고객사 온보딩 위저드 (4단계, 샘플 통화 시뮬레이션)
        • 고객 관리 콘솔 (기본 대시보드, 봇 설정, 문서 업로드)

        📍 Phase 2 (확장, M4~M6, 3개월)

        목표: 고급 기능 + 고객 만족도 향상

        • 감정 분석 + 에스컬레이션 자동화
        • 통화 자동 요약 + CRM 연동 (Salesforce, HubSpot 기본 API)
        • 아웃바운드 캠페인 기능 (프로모션/설문 전화)
        • LLMOps 대시보드 (프롬프트 A/B 테스트, 응답 품질 추적)
        • 고급 권한 관리 (팀 기반 RBAC)

        📍 Future (성숙화, M7+)

        목표: 경쟁 우위 확보 + 성과 기반 과금

        • 도메인별 특화 모델 (금융 규정, 이커머스 물류, 보험 청구)
        • 성과 기반 과금 자동 미터링 (Resolved Call 추적)
        • 마켓플레이스 (CRM 플러그인, 업체별 API 확장)
        • AI 모니터링 (편향 감지, 규정 위반 알림)

        6. 성공 지표 (KPI)

        비즈니스 메트릭

        지표 목표 측정 방법 기대 효과
        자기서비스 해결률 (SSR) 60% 이상 에스컬레이션 없이 완결된 통화 / 전체 통화 인력 30% 감소 (연 3억 절감)
        평균 처리 시간 (AHT) 23% 이상 단축 통화 시작~종료 + ACW 평균 일일 처리량 30% 증가
        첫 통화 해결률 (FCR) 70% 이상 재접촉 없이 종결된 비율 고객 만족도 +15%
        온보딩 TTF 3일 이내 가입~첫 실제 통화까지 소요일 판매 주기 단축, 고객 만족
        콜 오류율 1% 이하 오류 통화 / 전체 통화 규제 준수 (금융) 달성

        제품 메트릭

        • DAU/WAU: 고객사당 평균 일일/주간 활성 상담원 수 추적
        • 기능 채택률: Agent Assist, 감정분석 등 고급 기능 사용 비율
        • API 성공률: Closed-loop 태스크 성공 비율 (주소변경, 환불 등)
        • 평균 세션 길이: 통화당 토큰 사용량 (비용 최적화 지표)

        조직 메트릭

        • NRR(Net Revenue Retention): 기존 고객의 갱신 + 추가 구매율 (SaaS 건전성)
        • CSAT/NPS: 고객 만족도 (목표: CSAT 4.5/5, NPS 50+)
        • 고객 이탈률: 월간 churning rate (목표: 2% 이하)
