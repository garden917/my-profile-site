📌 과제 개요

🎯 목표

기존 구축형(SI) AICC 솔루션을 범용 멀티테넌트 SaaS 제품으로 전환하기 위한 완전한 기획

🎁 기존 자산

  • 보이스봇 코어 엔진
  • Elasticsearch + Neo4j RAG
  • React AdminPanel

🌍 타겟 시장

  • 금융: 은행, 보험 (규제 민감)
  • 이커머스: 온라인 쇼핑 (속도 중심)
  • 전자: 고객지원 (기술 중심)

⏱️ 타임라인

  • Phase 0: M1~M2 (기반 정비)
  • Phase 1: M3 (MVP 출시)
  • Phase 2: M4~M6 (확장)
  • Phase 3: M7~M10 (성숙화)

📑 산출물 목록

1️⃣ 제품 기획서 (PRD)

제품 비전, 타겟 고객, 핵심 기능, 자산 활용 계획, 우선순위, KPI

📄 6개 섹션 • 읽기 시간: 15분
2️⃣ 화면 설계 (Wireframes)

보이스봇 관리 콘솔, Agent Assist 화면, 고객사 온보딩 위저드 (3개 화면)

🎨 3개 화면 • 읽기 시간: 10분
3️⃣ 제품 로드맵

4 Phase 배포 계획, 타임라인, 리소스, 의존성 및 리스크 관리

📊 10개월 계획 • 읽기 시간: 12분
4️⃣ 가격 전략

구축형/SaaS형 가격 모델, 성과 기반 과금, 경쟁력 포지셔닝

💰 3가지 모델 • 읽기 시간: 10분

⚡ 빠른 참조

🎯 핵심 KPI
  • 자기서비스율: 60%+
  • AHT: -23% 이상
  • FCR: 70%+
  • 온보딩 TTF: 3일
💰 기본 가격
  • Starter: 150만원/월
  • Business: 500만원/월
  • Enterprise: 협의
  • 성과기반: 기본료+건당
📅 타임라인
  • Phase 0: 2개월
  • Phase 1 MVP: 1개월
  • Phase 2: 3개월
  • Phase 3: 4개월

🎲 주요 의사결정 사항

결정 항목 선택안 추천 방향 근거
제품 포지셔닝 SI 유지 vs SaaS 전환 SaaS 중심 시장 트렌드, NRR 우위, 스케일 가능
초기 타겟 금융 vs 이커머스 이커머스 우선 빠른 온보딩, 스파이크 니즈, 참조 고객 확보 용이
신규 개발 우선 Agent Assist vs 다른 기능 Agent Assist 상담원 생산성 핵심, 마진 개선 (프리미엄)
멀티테넌시 공유스키마 vs 전용DB 하이브리드 SMB는 공유, Enterprise는 전용 (비용-보안 균형)
가격 모델 정액제만 vs 성과기반 단계적 전환 초기는 정액, Phase 2부터 성과기반 병행 (고객 성숙도 고려)

⚠️ 주요 리스크 & 대응

🔴 High Priority

  • 멀티테넌시 보안: 데이터 유출 리스크 → 외부 보안감사 필수
  • 초기 고객 확보: 마케팅 리소스 부족 → 조기 투입
  • 감정분석 정확도: 도메인별 차이 → 파일롯 기반 미세조정

🟡 Medium Priority

  • CRM 연동 복잡도: 통합 변수 많음 → 기본 API만 MVP 포함
  • 성과기반 과금 구현: 미터링 로직 복잡 → 전담팀 구성
  • 리소스 확보: MLOps 인재 부족 → 채용 조기 시작
